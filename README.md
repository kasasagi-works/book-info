# 書籍情報取得ツール

ISBNを入力すると [openBD API](https://openbd.jp/) から書籍情報を取得し、note記事に貼り付けられるマークダウンテキストを生成するWebツールです。

## 機能

- ISBNから書籍情報（書名・出版者・出版年月・著者名）を取得
- note記事用のマークダウンテキストを自動生成
- ワンクリックでクリップボードにコピー

## 使い方

1. ISBNを入力欄に入力する（ハイフンあり・なし どちらも可）
2. **取得** ボタンを押す（またはEnterキー）
3. 生成されたマークダウンテキストをコピーしてnote記事に貼り付ける

**出力例：**

```
ISBN: **9784815633349**
書名: **ぼくはこんなふうに本を読んできた**
出版者: **ＳＢクリエイティブ**
出版年月: **202508**
著者名: **池上彰**
```

## セットアップ

バックエンド不要。`book-info.html` をレンタルサーバーにアップロードするだけで動作します。

```
your-server/
└── book-info.html
```
